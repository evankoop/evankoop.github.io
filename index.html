Evan's Virtual Baseball Cards

Evan's Virtual Baseball Cards is a fullstack application that allows users to make their own virtual baseball card.
It includes two pages, one for making a card and another for viewing their collection. Users can input a player
name, team, position, and image URL to create their card. Users can also delete cards in the collection page.

Technologies Used:

JavaScript, PostreSQL, Express, Axios, Sequelize, Cors, JSON, Postman, HTML, CSS
